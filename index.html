HTML Semantik

Tag Name Description
<header> Header Kopfbereich einer Webseite, der oft den Seitentitel und die Hauptnavigation enthält.
<nav> Navigation (Nav) Nutzung zur Auszeichnung von Navigationsmenüs und -links, die die Benutzer durch die Website führen.
<main> Main Markiert den Hauptinhalt einer Webseite und sollte nur einmal pro Seite verwendet werden.
<article> Article Verwendung für eigenständige Inhalte wie Blogposts, Nachrichtenartikel oder Forenbeiträge.
<section> Section Semantische Gruppierung von Inhalten zur Strukturierung von Abschnitten auf der Seite.
<aside> Aside Enthält Inhalte, die nicht direkt mit dem Hauptinhalt in Verbindung stehen, wie z.B. Seitenspalten oder Anzeigen.
<footer> Footer Markiert den Fußbereich einer Seite und enthält oft Copyright-Informationen, Kontaktinformationen oder Links zu sozialen Medien.
<address> Address Wird verwendet, um Kontaktinformationen im Footer zu kennzeichnen, wie die physische Adresse des Unternehmens oder E-Mail-Adresse für Kontaktzwecke.
<p> Paragraph (P) Verwendet zur Strukturierung von Absätzen und Textinhalt.
<div> Division (Div) Allgemeines Container-Element zur Strukturierung und Gestaltung von Inhalten.
<span> Span Inline-Element zur Anwendung von Stilen auf Text innerhalb eines Satzes oder Absatzes.
<em> Emphasis (Em) Betonung von Textinhalten, standardmäßig kursiv dargestellt.
<br> Line Break (Br) Erzeugt einen Zeilenumbruch innerhalb des Textes oder Inhalts.
<abbr> abbreviation Beispiel steht "HTML" für "Hypertext Markup Language". Das abbr-Element enthält das Attribut title, das den vollständigen Text der Abkürzung oder des Akronyms enthält.
