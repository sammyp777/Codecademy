block
  • Block-Elemente nehmen die gesamte verfügbare Breite ein und beginnen auf einer neuen Zeile
  • Sie können Höhe und Breite festlegen, Rand und Padding hinzufügen.
  • Block-Elemente können nicht nebeneinander angezeigt werden.

<em>		 inline
  • Inline-Elemente nehmen nur so viel Breite ein, wie für ihren Inhalt erforderlich ist, und bleiben in derselben Zeile wie der vorherige oder der nachfolgende Inhalt.
  • Sie können keine Höhe und Breite festlegen oder vertikalen Abstand hinzufügen.
  • Inline-Elemente können nebeneinander angezeigt werden.
inline-block
  • Sie nehmen nur so viel Breite wie nötig ein, können jedoch Höhe, Breite, Rand und Padding festgelegt werden.
  • Kombiniert Eigenschaften von Inline- und Block-Elementen.
  • Sie verhalten sich wie Inline-Elemente und können nebeneinander angezeigt werden, ermöglichen jedoch das Einstellen von Höhe und Breite.
flex
  • Dies ermöglicht die Verwendung von Flexbox-Layout. Mit Flexbox können Sie Elemente in einer Zeile oder Spalte anordnen und flexibel anpassen.
  • Flexbox bietet leistungsstarke Möglichkeiten zur Layout-Gestaltung.
grid
  • Dies ermöglicht die Verwendung von Grid-Layout. Grid-Layout bietet eine 2D-Anordnung von Elementen in Zeilen und Spalten, was für komplexe Layouts nützlich ist.

#ID .CLASS & SELECTOR

Tag Name Description
.class
#id
selector		 CSS
  • ID-Selektoren: IDs sind eindeutig in einer HTML-Seite und sollten daher nur für ein einziges Element verwendet werden. ID-Selektoren haben eine hohe Priorität in CSS, was bedeutet, dass Stile, die mit ID-Selektoren definiert sind, stärker sind als Stile, die mit Klassen oder Elementselektoren definiert sind.
  • Klassen-Selektoren: Klassen können auf mehreren Elementen verwendet werden, und sie sind eine gute Möglichkeit, um Stile auf Gruppen von Elementen anzuwenden. Klassen haben eine geringere Priorität als IDs, aber eine höhere Priorität als Elementselektoren.
  • Elementselektoren: Elementselektoren wählen alle Instanzen eines bestimmten HTML-Elements aus und haben die niedrigste Priorität. Sie werden üblicherweise verwendet, um Standardstile für Elemente festzulegen.

overflow-y

Property Tag Description
overflow-y: auto; Overflow-y Overflow-y ist eine CSS-Eigenschaft, die auf Container-Elemente angewendet wird, um das Verhalten des vertikalen Scrollens innerhalb des Containers zu steuern.
auto Auto Wenn der Inhalt im Container die verfügbare vertikale Höhe überschreitet, wird ein vertikaler Scrollbalken angezeigt, sodass Benutzer durch den Inhalt scrollen können. Wenn der Inhalt die verfügbare Höhe nicht überschreitet, wird kein Scrollbalken angezeigt.
scroll Scroll In diesem Fall wird immer ein vertikaler Scrollbalken angezeigt, unabhängig davon, ob der Inhalt die verfügbare Höhe überschreitet oder nicht. Wenn der Inhalt die verfügbare Höhe nicht überschreitet, ist der Scrollbalken deaktiviert.
visible Sichtbar Hier wird kein Scrollbalken angezeigt, selbst wenn der Inhalt die verfügbare Höhe des Containers überschreitet. Der Inhalt wird einfach über den Container hinaus angezeigt, was dazu führen kann, dass Teile des Inhalts unsichtbar sind.
overflow-y CSS-Eigenschaft Diese CSS-Eigenschaft steuert das Scrollverhalten innerhalb eines Containers und dient zur Anzeige von vertikalem Scrollen, wenn der Inhalt die verfügbare Höhe überschreitet.
display CSS-Eigenschaft Im Gegensatz zu overflow-y wird die "display"-Eigenschaft verwendet, um das Darstellungsverhalten von HTML-Elementen selbst zu steuern, einschließlich ihrer Positionierung, Art (Block, Inline, usw.), Layout-Verhalten und mehr.

CSS Box Model

Property Name Description
width Box Width Breite des Inhaltsbereichs (content) eines Elements, ohne Rand und Padding.
height Box Height Höhe des Inhaltsbereichs (content) eines Elements, ohne Rand und Padding.
padding Padding Der Abstand zwischen dem Inhalt und dem Rand eines Elements.
border Border Die Linie, die den Rand eines Elements umgibt.
margin Margin Der äußere Abstand eines Elements, der den Raum zwischen den Rändern von Elementen festlegt.
.element { box-sizing: border-box;} Box-Sizing Die Eigenschaft, die festlegt, wie die Größe eines Elements berechnet wird, entweder als "content-box" (nur Inhalt), "padding-box" (Inhalt plus Padding) oder "border-box" (Inhalt plus Padding plus Border).

CSS Text Styling & Design

Tag Bezeichnung Beschreibung
a {text-decoration: underline} Text Decoration Steuert die Dekoration von Text, z.B., "underline" (Unterstreichung), "line-through" (Durchgestrichen).
a:link, a:visited, a:hover, a:active Link Styling (Pseudo-Klassen) CSS-Pseudo-Klassen zur Definition von Stilen für Links in verschiedenen Zuständen: unbesucht, besucht, Mauszeiger darüber, aktiv.
href="link" title="Tooltip" Tooltip Ein Tooltip ist ein Informationsfeld, das beim Überfahren eines Elements mit der Maus angezeigt wird.
href="link" target="_blank" link öffnen Beim klicken des Links öffnet sich ein neuer Tab.
a:link {}, :visited, :hover, :active, :focus, Pseudo-Klassen Beachte die Reihenfolge der Tags in CSS
CSS-Pseudo-Klassen, die Elemente in verschiedenen Zuständen ansprechen, wie Mauszeiger darüber (:hover), Mauszeiger aktiv (:active), oder Fokussierung (:focus).
transition: margin-right 2s, color 1s; Transition CSS-Übergangseigenschaften, die die Animation von Elementen ermöglichen, z.B., "margin-right 2s" (2 Sekunden Verzögerung), "color 1s" (1 Sekunde Verzögerung). Immer nach Pseudoclassen!
a {cursor: pointer} :help, :wait, :crosshair, :not-allowed, :zoom-in Cursor Properties Steuert das Erscheinungsbild des Mauszeigers, wenn er sich über einem Element befindet, z.B., "pointer" (Hand), "text" (Textcursor).
.skeuomorphic:active {z.B. Margin Change, box-shadow, background, ...} Skeuomorphismus 3D (Button Element) Ein Designstil, bei dem Elemente realen Gegenständen ähneln, mit realistischen Texturen und Schatten.
.skeuomorphic:visited {der Button hebt sich nicht hervor} Flat Design 2D (Button Element) Ein minimalistischer Designstil, der auf Schatten und realistische Elemente verzichtet, um eine flache, zweidimensionale Ästhetik zu erzielen.
letter-spacing, word-spacing Letter & Word Spacing Erlaubt die Anpassung des Abstands zwischen Buchstaben (letter-spacing) oder Wörtern (word-spacing) in Texten.
text-transform: uppercase; Text Transform Steuert die Groß- und Kleinschreibung von Text, z.B., "uppercase" (Großbuchstaben), "lowercase" (Kleinbuchstaben), "capitalize" (Jeder Anfangsbuchstabe wird groß geschrieben).
