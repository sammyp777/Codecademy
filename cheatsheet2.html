CSS Box Model

Tag Name Description
width Box Width Breite des Inhaltsbereichs (content) eines Elements, ohne Rand und Padding.
height Box Height Höhe des Inhaltsbereichs (content) eines Elements, ohne Rand und Padding.
padding Padding Der Abstand zwischen dem Inhalt und dem Rand eines Elements.
border Border Die Linie, die den Rand eines Elements umgibt.
margin Margin Der äußere Abstand eines Elements, der den Raum zwischen den Rändern von Elementen festlegt.
.element { box-sizing: border-box;} Box-Sizing Die Eigenschaft, die festlegt, wie die Größe eines Elements berechnet wird, entweder als "content-box" (nur Inhalt), "padding-box" (Inhalt plus Padding) oder "border-box" (Inhalt plus Padding plus Border).

block
  • Block-Elemente nehmen die gesamte verfügbare Breite ein und beginnen auf einer neuen Zeile
  • Sie können Höhe und Breite festlegen, Rand und Padding hinzufügen.
  • Block-Elemente können nicht nebeneinander angezeigt werden.
<span>
<a>
<strong>
<em>		 inline
  • Inline-Elemente nehmen nur so viel Breite ein, wie für ihren Inhalt erforderlich ist, und bleiben in derselben Zeile wie der vorherige oder der nachfolgende Inhalt.
  • Sie können keine Höhe und Breite festlegen oder vertikalen Abstand hinzufügen.
  • Inline-Elemente können nebeneinander angezeigt werden.
inline-block
  • Sie nehmen nur so viel Breite wie nötig ein, können jedoch Höhe, Breite, Rand und Padding festgelegt werden.
  • Kombiniert Eigenschaften von Inline- und Block-Elementen.
  • Sie verhalten sich wie Inline-Elemente und können nebeneinander angezeigt werden, ermöglichen jedoch das Einstellen von Höhe und Breite.
flex
  • Dies ermöglicht die Verwendung von Flexbox-Layout. Mit Flexbox können Sie Elemente in einer Zeile oder Spalte anordnen und flexibel anpassen.
  • Flexbox bietet leistungsstarke Möglichkeiten zur Layout-Gestaltung.
grid
  • Dies ermöglicht die Verwendung von Grid-Layout. Grid-Layout bietet eine 2D-Anordnung von Elementen in Zeilen und Spalten, was für komplexe Layouts nützlich ist.
<position> position Die position-Eigenschaft ermöglicht es dir, die Position eines Elements auf einer Webseite festzulegen. Je nach Wert können Elemente relativ, absolut, fixiert oder klebrig positioniert werden, was ihre Positionierung auf der Seite beeinflusst.
<relative> relative Wenn die position-Eigenschaft auf "relative" gesetzt wird, ist die Position eines Elements relativ zu seiner Standardposition auf der Seite. Es kann verschoben werden, ohne den normalen Fluss des Dokuments zu beeinflussen.
<absolute> absolute Bei "absolute" ist die Position eines Elements relativ zum nächsten positionierten Elternelement. Es kann an beliebiger Stelle auf der Webseite verankert werden, bewegt sich jedoch immer mit dem Rest des Dokuments beim Scrollen.
<fixed> fixed "Fixed" ermöglicht es einem Element, an einer beliebigen Stelle auf der Webseite verankert zu bleiben, sodass es immer sichtbar bleibt, unabhängig von Scrollbewegungen.
<sticky> sticky "Sticky" ermöglicht es einem Element, sich in einem definierten Abstand zu seiner übergeordneten Containerposition zu verhalten. Es bleibt sichtbar, bis der Benutzer den Container scrollt, dann verhält es sich wie "fixed".
<z-index> z-index Der z-index eines Elements gibt an, wie weit vorne oder hinten ein Element auf der Seite erscheint, wenn es mit anderen Elementen überlappt.
<display> display Die display-Eigenschaft ermöglicht die Steuerung des vertikalen und horizontalen Flusses eines Elements in einem Dokument. Sie kann Werte wie "inline", "block" oder "inline-block" haben, um das Verhalten des Elements zu ändern.
<float> float Die float-Eigenschaft kann Elemente so weit wie möglich nach links oder rechts auf einer Webseite bewegen. Dies wird häufig für Layoutzwecke verwendet, um Elemente nebeneinander anzuordnen.
<clear> clear Die clear-Eigenschaft wird verwendet, um das Entfernen von Elementen auf der linken, rechten Seite oder auf beiden Seiten eines zuvor geschwimmten Elements zu steuern.
<box model> Box-Modell Das Box-Modell beschreibt, wie Elemente auf einer Webseite gerendert werden. Es umfasst die Eigenschaften von Inhalt, Padding, Rand und Rahmen, die die Gesamtabmessungen eines Elements beeinflussen.

CSS Position Properties

Tag Name Description
static Position Standardpositionierung (keine Anpassung)
relative Position Position relativ zum normalen Fluss, Verschiebung mit Top, Right, Bottom, Left
absolute Position Position relativ zum nächsten positionierten Elternelement, Verschiebung mit Top, Right, Bottom, Left
fixed Position Position relativ zum Viewport, bleibt sichtbar, während Sie scrollen
sticky Position Position relativ zum Viewport bis zu einer bestimmten Bildlaufposition, dann verhält es sich wie 'fixed'

Flexbox

Tag Name Description
justify-content CSS justify-content wird in erster Linie für die Ausrichtung und Anordnung von Elementen in Flexbox-Containern verwendet. Sie wirkt sich nicht direkt auf Text innerhalb von Elementen aus. justify-content steuert die horizontale (entlang der Hauptachse) Verteilung von Flexelementen im Container. Es gibt verschiedene Werte für justify-content, die beeinflussen, wie die Flexelemente im Container ausgerichtet werden.
Hauptachse CSS In einem Flexbox-Container ist die Hauptachse die Hauptausrichtung, in der die Flexelemente angeordnet sind. Standardmäßig verläuft die Hauptachse horizontal von links nach rechts.
display: flex; Flexbox-Container Mit "display: flex;" wird ein Element zu einem Blockcontainer mit Flexelementen im Inneren. Dies ist die Grundlage für Flexbox-Layouts.
display: inline-flex; Inline Flexbox-Container Erlaubt es, mehrere Flexcontainer in einer Zeile anzuordnen.
justify-content; Horizontale Anordnung Verwendet, um Elemente entlang der Hauptachse zu positionieren (z. B. horizontal auszurichten).
align-items; Vertikale Anordnung Verwendet, um Elemente entlang der Nebenachse zu positionieren (z. B. vertikal auszurichten).
flex-grow; Flexibles Wachstum Spezifiziert, wie viel Platz und in welchem Verhältnis Flexelemente entlang der Hauptachse absorbieren.
flex-shrink; Flexibles Schrumpfen Spezifiziert, wie viel Flexelemente und in welchem Verhältnis sie entlang der Hauptachse schrumpfen.
flex-basis; Anfangsgröße Definiert die anfängliche Größe eines Elements, wenn es mit flex-grow und/oder flex-shrink gestaltet wird.
flex; Flex-Shorthand Ermöglicht die gleichzeitige Festlegung von flex-grow, flex-shrink und flex-basis in einer Deklaration.
flex-wrap; Umbruch Bestimmt, ob Elemente entlang der Nebenachse verschoben werden sollen, wenn der Flexcontainer nicht ausreichend Platz bietet.
align-content; Anordnung von Zeilen Verwendet, um Zeilen entlang der Nebenachse zu positionieren, insbesondere in mehrzeiligen Flexcontainern.
flex-direction; Richtung Legt die Haupt- und Nebenachse des Flexcontainers fest.
flex-flow; Flexbox-Fluss Ermöglicht die Festlegung von flex-wrap und flex-direction in einer einzigen Deklaration.
Verschachtelung von Flexcontainern; Flexbox-Verschachtelung Durch Deklarieren von "display: flex;" oder "display: inline-flex;" für Kinder von Flexcontainern können verschachtelte Flexcontainer erstellt werden.
